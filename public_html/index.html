· PISCO BOX ·

Entorno de desarrollo LAMP basado en Debian Bullseye con Apache, PHP 8.3 y MariaDB

Versión Actual: 0.1.0

Comenzar

Para iniciar tu entorno de desarrollo con Pisco Box:

$ vagrant up

Con la máquina virtual funcionando podrás hacer login:

$ vagrant ssh
Nota: Asegúrate de tener instalado Vagrant y VirtualBox/Parallels/VMware antes de comenzar.

Acceso Web

El sitio por defecto estará disponible en:

# URL principal
http://localhost:8080

# Si configuraste un nombre de host local
http://piscobox.test
¿Cómo configurar un nombre de host local? sigue estas instrucciones para crear un nombre de host en tu computadora local: editando el fichero /etc/hosts

Credenciales MySQL/MariaDB

Usuario: piscoboxuser

Contraseña: DevPassword123

Host: localhost

Para conectarte a MariaDB desde la línea de comandos:

# Como usuario root
$ sudo mysql
# O lo que es igual
$ sudo mariadb

# Como usuario de base de datos desde línea de comandos:
$ mariadb -u piscoboxuser -p
# Cuando te lo pida, ingresa la contraseña: root

O desde tu aplicación PHP:

$mysqli = new mysqli("localhost", "piscoboxuser", "DevPassword123", "piscoboxdb");

Características Principales

  • Sistema Operativo: Debian Bullseye 64-bit
  • Herramientas: wget, curl, gnupg, lsb-release, ca-certificates, net-tools, git, vim
  • Zona Horaria: UTC
  • Configuración Regional: UTF-8
  • Servidor Web: Apache 2 con módulos: proxy_fcgi, setenvif, rewrite
  • Directorio Web: Configurado y listo para usar
  • PHP: 8.3 con PHP-FPM
  • Extensiones PHP: bcmath, bz2, cgi, gd, imap, intl, mbstring, pspell, tidy, xmlrpc, zip
  • Composer: Versión 2.x
  • Base de Datos: MariaDB Server & Client

Estructura del Proyecto

Así se ve la estructura de directorios en tu máquina local:

# Directorio de tu proyecto (máquina local)
/ruta/a/tu/piscobox
├── doc/ # Documentación y colaboración
├── extra_data/ # Datos extras, backups
├── provision/ # Scripts de aprovisionamiento
├── public_html/ # Document root accesible en web
├── utils/ # Plantillas de flujo de trabajo y otros
├── LICENSE # Licencia de Software
├── README.md # Bienvenida, instrucciones básicas
├── Vagrantfile # Configuración de Pisco Box
└── ... # Otros archivos de tu proyecto

# Directorio sincronizado en la VM
/var/www/html ← Sincronizado con public_html/
Importante: Algunos directorios como public_html/ y provision/files/ se sincronizan con la máquina virtual. Lo que significa que los cambios que realices en los archivos de tu máquina local se reflejarán inmediatamente en la máquina virtual.

Comandos Útiles

# Reiniciar Apache
$ sudo systemctl restart apache2

# Reiniciar PHP-FPM
$ sudo systemctl restart php8.3-fpm

# Ver logs de Apache
$ sudo tail -f /var/log/apache2/error.log
